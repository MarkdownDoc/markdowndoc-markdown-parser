## Getting Started:

1. You need to have npm installed (it comes with node)
2. Clone this repository: `git clone git@github.com:markdowndoc/markdowndoc-markdown-parser.git`
3. Install dependencies: `npm install`

## Install

~~~
npm install
bower install
grunt
~~~
